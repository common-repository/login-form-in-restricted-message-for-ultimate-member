== Description ==

Ultimate member is one of the <a href="https://www.iworksolo.com/">most popular</a> membership plugin out there. By default, Ultimate Member doesn't support login form shortcode inside restricted access messages.
<a href="https://www.realestatetipsblog.com/">Real estate tips</a> brings this plugin which enables this feature so that you can <a href="https://www.yoursmallbusinessblog.com/">use your</a> login form shortcode inside restricted access messages.

== Installation ==

All you'll need to do to get this setup is:

1. Install and Activate the plugin, now use UM shortcode in the restrict message. 

That's it!
